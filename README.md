# docuflux Repository

> **Zentrale Dokumentation und Assets für die docuflux-Plattform**  
> KI-gestützte Dokumenten-Automatisierung für kleine und mittlere Unternehmen

## 🎯 Über docuflux

docuflux ist eine modulare KI-Plattform zur Automatisierung von Dokumenten-Workflows in KMU. Das erste Modul **RechnungsAI** automatisiert die Verarbeitung von Eingangsrechnungen durch intelligente Texterkennung und DATEV-Integration.

### Vision
Das Betriebssystem für sämtliche Dokumenten-Workflows in kleinen und mittleren Unternehmen werden.

### Mission
Unternehmen von manueller, repetitiver Dokumentenarbeit befreien, damit sie sich auf ihr Kerngeschäft konzentrieren können.
